Description

  ATSuccessStory provides a custom content type that allows to add 'Success Stories' to a Plone Site

Installation

  Place the ATSuccessStory folder inside the Products' folder of your zope instance and install using the "portal_quickinstaller"

Usage

  For using it you should read 'portlet_documentation' document included with the product
